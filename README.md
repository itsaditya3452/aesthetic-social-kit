# Aesthetic Social Kit

A zero-backend "Aesthetic Social Media Kit & Instant Status Engine" built with Next.js 14, Tailwind CSS, and lucide-react. Everything runs client-side — no database, no API keys.

## Features
- **Story Generator** — Birthday Bash / Minimalist Quote / Travel Vibe templates, rendered on an HTML5 canvas, exportable as HD JPG.
- **Script Flow Builder** — local pseudo-AI generator producing Hook / Retention Body / CTA shot-by-shot scripts.
- **Caption Studio** — aesthetic line breaks, Unicode bold/serif fonts, niche hashtag stacks, one-click copy.

## Run locally
```bash
npm install
npm run dev
```
Open http://localhost:3000

## Push to GitHub
```bash
git init
git add .
git commit -m "Aesthetic Social Kit"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO.git
git push -u origin main
```

## Deploy to Vercel
1. Go to https://vercel.com/new
2. Import the GitHub repo you just pushed.
3. Leave all settings as default (Next.js is auto-detected).
4. Click **Deploy**.

No environment variables are required.
